## 🌑 Overview

A high-performance, industry-grade personal portfolio built with **Next.js 15 (App Router)** and designed with a premium **"Deep Space Obsidian"** aesthetic. Built to showcase expertise in Backend Systems, AI Research, and Full-Stack Development.

The design philosophy: *dark, minimal, and brutally technical* — inspired by top creative developer portfolios.

---

## ✨ Features

### 🎨 Design & Aesthetics
- **Deep Space Obsidian** theme — pure black background with Electric Emerald & Violet accents
- **Grainy Film Texture** overlay for a high-end, editorial feel
- **Glassmorphism** cards with glowing hover states
- **Procedural Nebulas** — animated blue/indigo clouds drifting in the background
- **Live Starfield** — 150 twinkling stars rendered on HTML5 Canvas

### ⚡ Motion & Interactions (GSAP)
- **Lenis Smooth Scrolling** — momentum-based scroll with custom easing
- **Horizontal Projects Slider** — swipe-style card scroll driven by GSAP ScrollTrigger
- **Magnetic Buttons** — elements pull toward the cursor on hover
- **Staggered Text Reveals** — GSAP timeline entrance for Hero section
- **Liquid Custom Cursor** — trailing circle that morphs on interactive elements

### 🖥️ Interactive CLI Terminal
- Toggled via the **"About Me"** button in the Hero / Navbar
- Features a terminal **boot sequence** with typing animation
- Commands: `/work`, `/skills`, `/help`, `/clear`, `/exit`
- Styled to match the Obsidian theme

### 📬 Functional Backend
- **Contact Form** — validates and saves messages to **MongoDB Atlas**
- **Resume Download** — one-click PDF download
